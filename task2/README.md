# Task 2 — B2C консультант (RAG Public)

Мини-прототип функции агента для консультационных запросов физических лиц.

## Что реализовано

- Активация только для `client_type="B2C"` и `topic="consultation"`.
- Public RAG-логика (прототип): поиск по публичной базе знаний (чанки FAQ/Docs/тарификация).
- Приоритет текущего вопроса над историей сообщений при поиске.
- Дружелюбный ответ простым языком при достаточной релевантности.
- Безопасный fallback (`answer_found=false`) для:
  - персональных/индивидуальных кейсов;
  - отсутствия релевантного ответа в публичной базе;
  - неприменимого сценария.

## Test-driven подход

Сначала были добавлены тест-кейсы под целевое поведение, затем доработана логика скоринга/интентов для прохождения тестов.

## Запуск

```bash
python3 -m unittest task2/test_b2c_consultant.py
python3 task2/b2c_consultant.py
```

## Документация

Подробная функциональная документация: `task2/FUNCTIONAL_DOCUMENTATION.md`.

## Формат входа

```json
{
  "question_text": "Текст вопроса клиента",
  "dialog_context": {
    "messages_history": [
      {"role": "client", "text": "Предыдущее сообщение клиента"},
      {"role": "agent", "text": "Предыдущий ответ агента"}
    ]
  },
  "client_type": "B2C",
  "topic": "consultation"
}
```

## Формат выхода

```json
{
  "answer_text": "Готовый текст ответа для клиента",
  "answer_found": true
}
```
